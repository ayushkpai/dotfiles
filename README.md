# dotfiles

## Mac setup

- Open Terminal App

  ```
  $ xcode-select --install
  ```

- Install Homebrew

  ```
  $ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```

- Install Firefox

  ```
  $ brew install --cask firefox
  ```

- Install Ghostty

  ```
  $ brew install --cask ghostty
  ```

- Auto hide dock

  Open Settings > "Desktop & Dock" > "Enable Auto hide"

- Install Git

  ```
  $ brew install git
  ```
